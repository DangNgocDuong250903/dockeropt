üåç GeoIP Blocking Rules

Customize geographic blocking rules for Xray.

üìñ How to customize GeoIP rules:
  1. SSH to VPS: ssh your_user@34.150.92.64
  2. Edit config: sudo nano /usr/local/etc/xray/config.json
  3. Find the "routing" section
  4. Add your rules (see examples below)
  5. Restart Xray: sudo systemctl restart xray

Example Rules:

{
  "routing": {
    "rules": [
      {
        "type": "field",
        "ip": ["geoip:cn"],
        "outboundTag": "block"
      },
      {
        "type": "field",
        "ip": ["geoip:private"],
        "outboundTag": "direct"
      },
      {
        "type": "field",
        "domain": ["geosite:category-ads-all"],
        "outboundTag": "block"
      }
    ]
  }
}

Common GeoIP Codes:

  • geoip:cn - China
  • geoip:us - United States
  • geoip:ru - Russia
  • geoip:vn - Vietnam
  • geoip:private - Private IPs

Common GeoSite Categories:

  • geosite:category-ads-all - All ads
  • geosite:google - Google domains
  • geosite:facebook - Facebook domains
  • geosite:netflix - Netflix domains

üîß Useful Commands

Service Management:

sudo systemctl restart xray-monitor    # Restart dashboard
sudo systemctl restart xray             # Restart proxy
sudo systemctl status xray-monitor      # Check dashboard status
sudo systemctl status xray              # Check proxy status

View Logs:

sudo journalctl -u xray-monitor -f     # Dashboard logs
sudo journalctl -u xray -f              # Xray logs
sudo journalctl -u bandwidth-tracker -f # Bandwidth tracker
sudo journalctl -u traffic-monitor -f   # Traffic monitor

Config Files:

/opt/xray-monitor/monitoring-dashboard.py  # Dashboard code
/usr/local/etc/xray/config.json             # Xray config
/opt/xray-monitor/users.db                  # Users database
/opt/xray-monitor/bandwidth.db              # Bandwidth history
